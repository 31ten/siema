Playo Lightweight and simple carousel in pure JavaScript

Building another grid system or slider in 2016 is probably stupid idea. Thats why I built one. It is very lightweight (just 1kb gziped) and comes with no dependencies. It comes with no styling and no crazy functionality. As Brad Frost said "do that shit yourself". If you need some more complex tool I highly recommend amazing Flickity by David DeSandro or Swiper. By the way - I built a grid system too :)

Installation

A little bit of markup needed.

Use npm / yarn or inject minifed script on ypur page manually.
