Siema Lightweight and simple carousel in pure JavaScript

Building another grid system or slider in 2016 sounds like a stupid idea. Thats why I built one. It is very lightweight (just 1kb gziped) and comes with no dependencies. It comes with no styling and no crazy functionality. As Brad Frost said "do that shit yourself". If you need some more complex tool I highly recommend Flickity by David DeSandro or Swiper. By the way - I built a grid system too :)

Installation

Setup is trivially easy. Let's add required markup.

<div class="siema">
  <div>Hi, I'm slide no 1</div>
  <div>Hi, I'm slide no 2</div>
  <div>Hi, I'm slide no 3</div>
  <div>Hi, I'm slide no 4</div>
</div>

If you are using module bundler like Webpack or Browserify...

npm i -S siema
import Siema from 'siema';
new Siema('.siema');

...or manually inject minified script to your website.

<script src="siema.js"></script>
<script>
  new Siema('.siema');
</script>

Options

Siema comes with few options that you can chnge by passing an object as a econ argument. Default values presented below. Hopefully all the names are sefl explained.

new Siema('.siema', {
  duration: 300,
  easing: 'ease-out',
  perPage: 1,
  startIndex: 0,
  draggable: true,
  threshold: 20
});
