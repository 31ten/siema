Siema Lightweight and simple carousel in pure JavaScript

Siema is a lightweight carousel plugin with no dependencies and no styling. As Brad Frost said once "do that shit yourself". Is 100% open source available on Github. Free to use on personal and commercial projects. Use it with ease with your module bundler or by manually injecting to your project.

Download Siema from Github

Installation

Setup is trivially easy. A little bit of markup...

<div class="siema">
  <div>Hi, I'm slide 1</div>
  <div>Hi, I'm slide 2</div>
  <div>Hi, I'm slide 3</div>
  <div>Hi, I'm slide 4</div>
</div>

If you are using module bundler like Webpack or Browserify...

npm i -S siema
import Siema from 'siema';
new Siema();

...or manually inject minified script to your website.

<script src="dist/siema.min.js"></script>
<script>
  new Siema();
</script>

API

As mentioned above, Siema doesn't come with many options. Thing that it comes with are few useful methods. Combine it with very basic JavaScript and voila!

  • next() - Go to next slide
  • prev() - Go to previous slide
  • goTo(index) - Go to specific slide
  • currentSlide - Index of active slide (read only)

const mySiema = new Siema();
document.querySelector('.prev').addEventListener('click', () => mySiema.prev());
document.querySelector('.next').addEventListener('click', () => mySiema.next());

Siema is so lightweight becaus it doesnt have any options that you can built by yourself in few seconds. Let's display two slides on page and enable auto play with interval of 3 seconds.

const mySiema = new Siema({
  perPage: 2,
});
document.querySelector('.prev').addEventListener('click', () => mySiema.prev());
document.querySelector('.next').addEventListener('click', () => mySiema.next());
setInterval(() => mySiema.next(), 3000);

OK you get it. Last one with some bouncy easing and arrow keys bound to next() and prev() method.

const mySiema = new Siema({
  easing: 'cubic-bezier(.09,.75,.4,1.93)',
  duration: 600,
});
document.querySelector('.prev').addEventListener('click', () => mySiema.prev());
document.querySelector('.next').addEventListener('click', () => mySiema.next());
document.addEventListener('keydown', (e) => {
  if (e.keyCode === 37) {
    mySiema.prev();
  }
  else if (e.keyCode === 39) {
    mySiema.next();
  }
});

Contributing

Siema suppose to provide a basic carousel tool and allow developer to extend it by available methods. It doesn't come with complex configuration and gazzylion options — I would like to keep it this way. If you need some more powerful library I recommend amazing Flickity by David DeSandro or Swiper by iDangero team.

Bug fixed more than welcone. If you have a feature suggestion please open an issue on Github. Before sending a pull reques make sure to use 'build' task please.

npm run build
