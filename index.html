Siema Lightweight and simple carousel in pure JavaScript

Building another grid system or slider in 2016 sounds like a stupid idea. Thats why I built one. It is very lightweight (just 1kb gziped) and comes with no dependencies. It comes with no styling and no crazy functionality. As Brad Frost said "do that shit yourself". If you need some more complex tool I highly recommend Flickity by David DeSandro or Swiper. By the way - I built a grid system too :)

Installation

Setup is trivially easy. Let's add markup.

<div class="siema">
  <div>Hi, I'm slide no 1</div>
  <div>Hi, I'm slide no 2</div>
  <div>Hi, I'm slide no 3</div>
  <div>Hi, I'm slide no 4</div>
</div>

If you are using module bundler like Webpack or Browserify...

npm i -S siema
import Siema from 'siema';
new Siema('.siema');

...or manually inject minified script to your website.

<script src="siema.js"></script>
<script>
  new Siema('.siema');
</script>

Options

Siema comes with few options that you can chnge by passing an object as a second argument. Default values presented below. Hopefully all the names are sefl explained.

new Siema('.siema', {
  duration: 300,
  easing: 'ease-out',
  perPage: 1,
  startIndex: 0,
  draggable: true,
  threshold: 20
});

API

As mentioned above, Siema doesn't come with crazy amount of options. Thing that it comes with is bunch useful methods. Combine it with very basic JavaScript and go crazy.

  • next() - Go to next slide
  • prev() - Go to previous slide
  • goTo(index) - Go to specific slide

Examples

Very basic carousel with next and previous buttons.

const mySiema = new Siema('.js-siema');
document.querySelector('.next').addEventListener('click', mySiema.next);
document.querySelector('.prev').addEventListener('click', mySiema.prev);
